# Taraji-website
A full-stack e-commerce web application for buying and selling furniture online. Built with PHP, MySQL, and JavaScript, this platform offers a smooth shopping experience for customers and a robust analytics portal for business owners.

Key Features:
Browse, search, and order furniture online.

Secure online payment integration.

Admin portal for managing products, orders, and customers.

Business analytics dashboard to track sales, revenue, and customer insights.

Technologies: PHP | MySQL | HTML | CSS | JavaScript

This project showcases end-to-end web development, including frontend design, backend logic, and database management.
